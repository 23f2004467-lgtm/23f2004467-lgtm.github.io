Silver Medal · Top 5%

AutoStream Agent

Live · LangGraph
Conversational lead-capture agent · 2025

A multi-turn LangGraph agent that converts product questions into qualified leads via a phase-driven state machine (browsing → qualifying → confirming → captured). FAISS-backed RAG gated to fire only on product-inquiry intent; tool-calling layer with Pydantic-structured output and JSON-mode fallback; lead-capture tool gated on three filled slots, regex-validated email, and explicit user confirmation. 14-test eval suite, sub-2-second cold start, full WhatsApp Business Cloud API deployment path documented.
